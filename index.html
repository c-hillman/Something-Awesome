Item ID:




Item IDs are calculated by reversing the last four hexadecimal bytes of
a decoded item chat links. Chat links can be found on an item's GW2 wiki page.
ie. https://wiki.guildwars2.com/wiki/<item name>
Chat links are encoded in base64, can can be decoded with various online tools.

For example, Twilight has the chat link [&AgHwdwAA]. Stripping the brackets
and the ampersand, you have AgHwdwAA. Decoding this yields 0x 02 01 F0 77 00 00.
Taking the last four bytes and reversing them gets 00 00 77 F0. Converting this
back to decimal yields its itemID, 30704.
